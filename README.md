# Bank Account Simulator (Java)

A console-based banking system built using Java and Object-Oriented Programming principles.

## Features

* Create new bank accounts
* Deposit money
* Withdraw money
* View current balance
* Transaction history tracking
* Savings account interest calculation
* Multiple account management

## Technologies Used

* Java
* Object-Oriented Programming (Inheritance, Encapsulation)
* ArrayList
* Scanner for user input

## Project Structure

BankSimulator
│
├── Account.java
├── SavingsAccount.java
├── Bank.java
└── Main.java

## How to Run

1. Compile the program:

javac *.java

2. Run the application:

java Main

## Example Operations

* Create account with initial deposit
* Login using account number
* Deposit or withdraw funds
* View transaction history
* Apply interest to savings account
